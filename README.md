🚲 Bike Sales Analysis Dashboard (Excel)

## 📌 Project Overview
This project involves a comprehensive analysis of customer data to identify key factors influencing bike purchases. By processing raw data into a dynamic Excel dashboard, this analysis uncovers trends related to income, age, commute distance, and regional demographics.

## 🛠️ Technical Workflow
The project was executed in three main phases:

1. **Data Cleaning & Preprocessing:**
    * Removed duplicate records to ensure data accuracy.
    * Standardized categorical values (e.g., converted "M/S" to "Married/Single").
    * Used **Nested IF functions** to create a new `Age Bracket` column, categorizing customers into "Adolescent," "Middle Age," and "Old."

2. **Data Analysis (Pivot Tables):**
    * Aggregated data to find correlations between **Average Income** and purchase status.
    * Analyzed **Commute Distance** patterns for bike buyers.
    * Segmented sales by **Region** and **Car Ownership**.

3. **Data Visualization (Dashboarding):**
    * Designed a professional **Dark Mode Dashboard**.
    * Integrated **Interactive Slicers** (Region, Education, Marital Status) to allow for real-time data filtering.

## 📊 Key Insights
* **Income Impact:** Male customers who purchased bikes have the highest average income (~$60k), suggesting bikes are often a lifestyle or high-end purchase in this dataset.
* **The "Middle Age" Market:** The vast majority of bike buyers fall into the **Middle Age** bracket, making them the primary target audience.
* **Commute Factor:** Customers living within **0-1 miles** of their workplace are significantly more likely to buy a bike compared to those with longer commutes.
* **Regional Performance:** **North America** represents the largest market share for bike sales among the analyzed regions.

## 🚀 How to Use the Project
1. Clone this repository or download the `.xlsx` file.
2. Open the file in **Microsoft Excel**.
3. Go to the **Dashboard** tab.
4. Use the slicers on the right to filter data and observe how demographics affect sales trends.
